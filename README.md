# agbj-ui
## Install
```shell
npm install agbj-ui -S
```

## Quick Start
``` javascript
import Vue from 'vue'
import Agbj from 'agbj-ui'

Vue.use(Agbj)

// or
import {
  agIcon
  // ...
} from 'agbj-ui'
Vue.component(Button.name, Button)
```
## Browser Support
Modern browsers and Internet Explorer 10+.
